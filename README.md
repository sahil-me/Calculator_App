# Calculator App

## Overview

The **Calculator App** is a simple yet powerful application that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. This project is developed using Java and provides a user-friendly interface for quick calculations.

## Features

- **User-friendly GUI**
- **Supports basic arithmetic operations:** Addition, subtraction, multiplication, and division
- **Error handling** for invalid inputs
- **Lightweight and efficient performance**

## Technologies Used

- **Programming Language:** Java
- **Development Tools:** IntelliJ IDEA
- **Libraries:** Java Swing (for GUI)

## Installation & Usage

### Clone the Repository
```sh
git clone https://github.com/sahil-me/Calculator_App.git
```

### Navigate to the Project Directory
```sh
cd Calculator_App
```

### Compile and Run the Application

#### If using an IDE:
- Open the project and run the `Main.java` file.

#### If using the command line:
```sh
javac Main.java
java Main
```
