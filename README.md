## Introduction

**Calculator App** is a desktop application developed in Java using the Swing GUI framework. It provides an intuitive interface for performing arithmetic calculations quickly and accurately.

The project demonstrates object-oriented programming principles, event-driven programming, graphical user interface development, and arithmetic computation using Java Swing.

---

## Features

- Clean and intuitive graphical user interface
- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- Decimal number support
- Large number calculations
- Input validation
- Error handling
- Lightweight desktop application
- Fast calculation execution

## Application Workflow

1️⃣ Launch the Calculator Application.

2️⃣ Enter numeric values using the keypad.

3️⃣ Select an arithmetic operation.

4️⃣ Press the **Equals (=)** button.

5️⃣ View the calculated result instantly.

6️⃣ Press **0** to clear the current input and start a new calculation.
